## 📌 Overview

The **Failure Intelligence System (FIS)** is designed to help users understand *why they fail* and *how to improve* using data-driven insights.

This system integrates Machine Learning, AI-based analysis, and interactive dashboards to create a complete learning and improvement ecosystem.

## 🧠 Core Features

### 🔍 Failure Analysis Engine
- Accepts user failure inputs (prep hours, confidence, description)
- Applies TF-IDF and feature engineering
- Uses K-Means clustering to detect:
  - Confidence Issue  
  - Knowledge Gap  
  - Preparation Problem  

---

### 📊 Intelligence Dashboard
- Failure Intelligence Score
- Risk prediction system
- Weekly growth tracking

---

### 📈 Advanced Analytics
- Failure type distribution (bar chart)
- Cluster distribution (pie chart)
- Performance analysis (scatter plots)
- Interactive filters with real-time updates

---

### 🎯 Improvement Plan Generator

- Difficulty scoring

- Streak system

### 📓 Reflection Journal

## 🏗️ System Architecture
User Input
↓
Data Module (Cleaning + TF-IDF)
↓
Feature Matrix
↓
ML Model (K-Means Clustering)
↓
Cluster Detection
↓
Microplan + AI Mentor
↓
Dashboard & Analytics
